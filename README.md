# dumgorf.net

Tools and wiki for the NEVER DIE GUY universe. Dice calculators, asset utilities, algorithm visualizers, NPC lore pages, and more.

**https://dumgorf.net**

## Stack

- React 19, TypeScript, Vite 7
- Material UI 7
- React Router 7

## Tools

Dice Calculator, Vectorize, Code-to-Figma, Algorithms Playground, RNG Playground, Loot Table Explorer, Asset Checker, Input Formatter, Motion Sandbox, Audio Utility, FFmpeg Tool, Stream Viewer, NPC Viewer, Chat Demo, Feedback Widget
